## Login Issue

**Sub-category:** Login Issue

### What to Check

If the error message states the user is disabled, re-enable the account through ICRM:

**ICRM Panel → Seller Info → Search by Company ID → Actions → View Details → Scroll down → Click "Active User"**

### Seller Reply
> Your user account has been re-enabled. Please try logging in again.

## Enable Partial COD

**Category:** Tech — Pre Assignment
**Sub-category:** Enable Partial COD

### What to Check

Enable Partial COD for the seller via ICRM. Make sure to also update the label type to **3** (or select the new label flow):

**ICRM Panel → Seller Info → Search by Company ID → Actions → View Details → Scroll down → Click "Enable Partial COD"**

Then set label type to `3` or select the new label flow option.

### Seller Reply
> Partial COD has been enabled for your account. Please check future orders — the Partial COD amount will be reflected in the order details and on the shipping label.

## Product Info

### Catalogue / Product Management

Sellers can add and manage their product catalogue in Shiprocket for inventory tracking, order mapping, and QC-enabled returns.

**Add a product:** Setup and Manage > Catalogue > All Products > + Add Product

**Add a category:** Setup and Manage > Catalogue > Categories > Add New Category

**Bulk product upload:** Catalogue > All Products > Bulk Upload > Download template > Fill data > Upload

### Packaging Master

Save package dimensions to autofill during order creation and avoid repeated manual entry.

**Path:** Setup and Manage > Packaging > Package Master

- Save packages with L × W × H (cm) and weight (kg).
- Map SKUs to packages.
- Upload image proofs to request volumetric weight freezing — prevents weight discrepancy for those packages.

### Rate Calculator

Check freight rates before creating an order.

**Path:** Tools > Rate Calculator

- Select shipment type (Forward or Return).
- Enter pickup pincode, delivery pincode, weight, and dimensions.
- View all available courier partners with rates, EDD, and services.
- Also available for **international** shipments.

### Reports

**Path:** Tools > Report > Shiprocket Reports tab

Download reports for orders, shipments, weight discrepancies, NDR, refunds, etc. Reports are sent to the registered email or available for direct download.

### Bulk Actions

**Path:** Tools > Bulk Actions

Enter Order IDs or Tracking Numbers and select action type:
- Print Label
- Print Invoice
- Print Manifest

### Activity Log

**Path:** Tools > Activity Log

Track uploaded files, bulk orders, errors, and recent bulk operations.

### Dashboard Overview

The Dashboard provides:
- Today's orders, pending pickups, in-transit shipments.
- COD remittance status.
- NDR / RTO summary.
- Financial metrics and wallet balance.
- Promotional banners and product highlights.

### Quick Actions (Top Navigation Bar)

From the top navigation bar's Quick Actions tab, sellers can:
- Add a new order
- Track a shipment
- Recharge wallet
- Check wallet balance (displayed next to the wallet icon)

### Upcoming Pickups (Home Page)

The Home page shows upcoming pickups scheduled for today and tomorrow. From here, sellers can:
- Download labels, invoices, and manifests for upcoming pickups.
- View pickup status and take action.

### Profile Widget (Top Right)

Shows: phone number, email ID, company ID, KAM details, current plan. Quick access to plan upgrade, referral program, and billing.

## How to Do on Seller Panel

### Add a Product to Catalogue

1. Go to **Setup and Manage > Catalogue** from the left menu.
2. Click **All Products**.
3. Click **+ Add Product** (or **Bulk Upload** for multiple products).
4. Fill in product name, SKU, price, HSN code, category, and other details.
5. Save.

### Save a Package in Package Master

1. Go to **Setup and Manage > Packaging > Package Master**.
2. Click **+ Add Package**.
3. Enter package name, dimensions (L × W × H in cm), and weight.
4. Optionally upload images for volumetric freeze.
5. Map SKUs if needed.
6. Save.

### Use the Rate Calculator

1. Go to **Tools > Rate Calculator**.
2. Select shipment type (Forward / Return / International).
3. Enter pickup pincode, delivery pincode, weight, and dimensions.
4. View all available courier options with rates, EDD, and services.

### Generate a Report

1. Go to **Tools > Report**.
2. Under **Shiprocket Reports** tab, find the desired report type.
3. Click **Download**.
4. Report is sent to your registered email or available for direct download.

### Check Upcoming Pickups

From the **Home page**, scroll to the **Your Upcoming Pickups** section to view and manage pickups scheduled for today and tomorrow.

### Access Profile and KAM Details

1. Click on the **Profile icon** (top right).
2. View: phone number, email ID, company ID, KAM name, KAM email, KAM phone.

## Mobile App — How to Do

### Manage Channels (Mobile)

1. Three-dots menu at bottom > **Channel Integration**.
2. View connected channels and add Shopify.

### Check Wallet Balance (Mobile)

Wallet balance is visible on the **Home screen** of the Shiprocket app next to the wallet icon.

### Set Courier Priority (Mobile)

Three-dots menu > **Settings > Courier Priority** > Select: Recommended, Cheapest, Fastest, Best Rated.

### Change Primary Pickup Address (Mobile)

Three-dots menu > **Settings > Pickup Address** > Swipe left on address > **Set Primary**.

### Access Profile (Mobile)

Tap the **Profile icon** (top right) to view and update your profile details including mobile number.

### Change Mobile Number (Mobile)

1. Open profile from top right corner.
2. Tap **phone number field > Edit Number**.
3. Enter new number.
4. Verify with OTP.

> Email address **cannot** be changed on the mobile app or web panel.

## Reports — Full Detail

### Available Report Types

**1. All Order Report**
- Tracks lifecycle of AWBs: zones, type (prepaid/COD), package dimensions, pickup schedules, ETD.
- Key insights: Payment mode, channel %, pickup location, status of AWB, estimated delivery date.
- Logic: AWB generated date within selected date range.

**2. NDR Report**
- NDR delivery through Shiprocket, response by attempts, delivery requests, attempts within 24–48 hours, final delivery.
- Helps optimize delivery process.

**3. Weight Discrepancy Report**
- Download from Billing > Weight Discrepancy > Download icon (top right). Report sent to registered email.

**4. COD Remittance Report**

**5. Refund Report**
- Returns > Return Received tab > Download Report > Refund Report.

### Scheduling Reports

- Reports can be scheduled to be sent via **WhatsApp, Email, or Webhook**.
- Maximum **10 actively scheduled reports** at a time.
- Maximum data range per report: **30 days**.
- Format: CSV or XLS.

**Path:** Tools > Reports > Schedule Report > Select report name, medium, frequency, date range, and time.

**Demo/Sample:** Tools > Reports > Schedule Report popup > Enter email/WhatsApp > Get Demo Message.

**View scheduled reports:** Tools > Reports > My Scheduled Report > Scheduled Report History.

### Rate Card

Tools > Rate Card — view courier-wise and zone-wise pricing with minimum weight and mode of transportation.

Three tabs: Forward rates, Reverse rates, Document rates.

To compare rates across plans: Use the plan dropdown (Lite, Basic, Advanced, Pro) on the Rate Card page. Filters available for shortlisting couriers by parameters.

### Activity Logs

Tools > Activity Logs — a log of all bulk actions, imports, and downloads in your account.

Direct link: https://app.shiprocket.in/seller/activity-logs

### Bulk Print Actions

Tools > Bulk Print Actions > Select Action Type (Print Label) — print up to **500 labels** in one go.

### Pincode Zone Mapping

Tools > Pincode Mapping Zone — check serviceability between pincodes and export full serviceability list.

**Export serviceable pincodes:** Tools > Pincode Mapping Zone > Pickup Zone Serviceability Sheet > Enter Pickup Pincode > Export (sent to registered email).

### Dashboard Data

Dashboard data is updated on **T-1 basis** — there may be a slight delay. Real-time data requests are not currently supported.

**Switch between Domestic and International dashboard:** Top left dropdown on Dashboard page.

## Product Info

### What is Shiprocket Checkout?

Shiprocket Checkout (formerly Fastrr) is a one-page checkout solution for e-commerce websites. It reduces checkout time by pre-filling customer addresses from Shiprocket's delivery database, displays EDD (Estimated Delivery Date), and provides tools to reduce RTO and increase conversions.

**Support:** checkoutsupport@shiprocket.com (10 AM–7 PM, Mon–Sat for calls; email support is 24/7)

**Website:** https://checkout.shiprocket.in/

### Address Pre-fill

- Shiprocket has delivered to 25+ crore customers. Address pre-fill rate: ~85–95%.
- When a customer verifies their phone via OTP at checkout, their address is auto-fetched from Shiprocket's database based on past deliveries.
- Only **delivered** addresses are pre-filled.
- Removing OTP step disables pre-fill — not recommended.

### Compatible Platforms

Shopify, WooCommerce, Magento, and Custom websites.

### Supported Payment Gateways

Cashfree, PayU, Snapmint, Simpl, Phonepe, Billdesk, Easebuzz, Cred, Razorpay, Setu, Decentro. Multiple gateways can be integrated simultaneously.

**Gateway failover:** If one gateway is down, checkout auto-switches to the secondary gateway — no payment downtime.

### Supported Payment Modes

UPI, Credit/Debit Cards, Netbanking, Wallets, BNPL, EMI.

### Does Not Support

- Paytm payment gateway.
- CCAvenue payment gateway.
- International shipping.
- Seller SMS notifications when orders are placed.
- COD to Prepaid and Aftersell on WooCommerce (Shopify only).

### How Checkout Reduces RTO

1. **OTP Verification** — verifies user phone number.
2. **Disable COD** — block COD for high-risk profiles (Very High or High risk).
3. **COD Charges** — add extra charge on COD to incentivize prepaid.
4. **Prepaid Discounts** — discount only on prepaid orders.
5. **Prepaid-only Discounts** — automatic or coupon-based, for prepaid methods only.
6. **Block RTO PINcodes** — no orders from specific pincodes.

### Discount Types Available

- Amount off product (fixed or percentage on specific products/collections)
- Amount off cart (fixed or percentage on total cart value)
- Bundle discounts (X products for Rs. Y, or discount on Y products)
- Tiered discounts (based on cart value or quantity)
- Bank offers (card-specific using PG Discount ID)
- Buy X Get Y
- Aftersell offers (on thank-you page)
- UTM-based discounts (for specific campaigns)
- Customer-specific coupon codes (by mobile number or first-time visitor)

> One coupon code can be combined with multiple automatic discounts. Two coupon codes cannot be combined.

### Delivery Methods

| Method | EDD |
|---|---|
| Rush | Less than 2 days |
| Express | 2–5 days |
| Standard | More than 5 days |

Different shipping fees can be charged per delivery method.

### AOV Increase Features

- **Aftersell** — show product offers on thank-you page; customer redirected back to checkout on click.
- **Prepaid discounts based on invoice value slab** (tiered).
- **Tiered discounts** on cart value or quantity.
- **Bundle discounts.**

### RTO Risk Blocking

COD can be blocked based on:
- Customer's risk profile: Very High or High risk.
- RTO tags (logistical intelligence from Shiprocket).
- Invoice value above a threshold.
- Number of COD orders in last X days.
- Product SKU or collection.
- Destination pincode.

### Tracking Integrations

GA4, Google Ads, Facebook Pixel, Moengage, Clevertap, Webenggage, Engage 360, Snapchat, Wigzo, Contlo, Aisensy, Kwikpass, Kwikchat.

### Loyalty Partners Supported

PopCoins, Flits, Easyrewardz, Capillarytech, Uloyal.io, Twidpay.

### Shopify Apps Listed by Shiprocket Checkout

- **Shiprocket: Smart COD** — custom rules to block COD and set shipping visibility on Shopify checkout.
- **Shiprocket: Smart Address** — address pre-fill with OTP verification on Shopify checkout.
- **Shiprocket Smart Cart** — slide cart with upsells, discounts, free gifts, progress bars, and milestone-based freebies.
- **Shiprocket Smart EDD** — real-time delivery dates using pincode validation and multi-warehouse intelligence.

### Billing

- No onboarding or integration fee.
- Charges: small percentage commission on invoice value of orders placed through checkout.
- Monthly invoices generated and downloadable from the Shiprocket dashboard (invoice section).

### Checkout Wallet

- Separate from Shiprocket shipping wallet — cannot be transferred between them.
- Checkout wallet balance **can** be transferred to bank account.
- Recharge directly from checkout dashboard (top right corner: Recharge Wallet).

### Reporting and Analytics

Dashboard sections:
- **Orders tab** — complete order details, payment details, discount codes.
- **Abandoned Carts** — pushed to Shopify in real time; visible under Checkout Dashboard > Abandoned Carts > Channel > Fastrr.
- **Pending Orders** — payment received but order not placed (Checkout Dashboard > Orders > Pending Orders).
- **Reports** — order details, refunds, checkout funnel performance. Max 30-day range. Sent to email via Report section.
- **Campaign Performance, Payment Success Rates, Discount Coupon Burn Rate, Checkout Funnel** — all available.

### User Access Levels

Admin, Finance, Marketing, Operations, Read-Only, Read-Write (customizable) — managed from Checkout Dashboard User Management.

### A/B Testing

Traffic can be split equally between two checkout solutions to compare performance metrics.

## How to Do on Checkout Dashboard

### Integrate a Payment Gateway

Settings > Payment > Add new gateway > Select gateway > Complete credentials.

### Set Prepaid Discount

Settings > Shipping Settings > Enter fixed or percentage discount under Prepaid Discount.

### Add COD Charges

Settings > Shipping Settings > Enter fixed or percentage under COD Charges.

### Set Shipping Charges by Delivery Method

Settings > Shipping Settings > Scroll to Shipping Charges > Enter amounts for Rush, Express, Standard.

### Block Non-Serviceable Pincodes

Settings > Shipping Settings > Non-Serviceable Pin Codes > Upload CSV or enter list.

### Create a Custom Shipping Rule

Settings > Shipping Settings > Custom Rules > Create Rule > Select Shipping Visibility > Set parameters (invoice value, pincodes, etc.).

### Create a Discount

Settings > Discounts > Create > Select discount type and method (automatic or coupon-based) > Configure parameters.

### Add GA4 Tracking

Settings > Tracking Info > GA4 tab > Enter Measurement ID.

### Add Facebook Pixel

Settings > Tracking Info > Facebook tab > Enter Pixel ID and Access Token (up to 5 Pixel IDs).

### Add Webhook

Settings > Webhooks > Add Webhook > Select Webhook Type > Enter API Endpoint > Select Stage > Add Headers > Save.

### Update Store Logo

Checkout Dashboard > Settings > General Settings > Update Logo.

### Enable UPI

Settings > Payment > Add new gateway > Select Payment Gateway > Select UPI (UPI must first be activated by payment gateway).

### Enable Gift Cards (Shopify)

Email checkoutsupport@shiprocket.com with domain name > Team activates within 12–24 working hours.

### View API Credentials (Shopify)

Shopify Dashboard > Settings > Apps & Sales Channel > Develop Apps > Select Fastrr Checkout > API Credentials.

### Enable Partial COD on Checkout

Email checkoutsupport@shiprocket.com > Team activates partial payment and changes label type to "Label 3" on Shiprocket platform.

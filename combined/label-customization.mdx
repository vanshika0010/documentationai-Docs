## Product Info

### Self-Serve Label Customization

Sellers can now design and control shipping labels directly from the Shiprocket dashboard — no manual intervention required.

**Access:** Settings > Company Setup > Labels

### What Can Be Customized (40+ Parameters)

| Category | Configurable Options |
|---|---|
| **Branding & Logo** | Show/Hide Shiprocket branding, brand logo, channel logo |
| **Support Details** | Show/Hide or edit customer support contact details |
| **Delivery & Pickup Details** | Toggle buyer/seller information, barcodes, routing codes, RTO details, GSTIN |
| **Order & Product Details** | Show/Hide SKUs, product names, prices, discounts, HSN, shipping charges, collectable amounts |
| **Miscellaneous** | Add/edit Notes, show invoice/E-Way Bill/order details |
| **Formatting Controls** | Adjust font styles (bold/italic/underline), trim long SKUs/product names, decide line items display |

### Default Character Limits

| Field | Limit |
|---|---|
| SKU | 10 characters |
| Product Name | 10 characters |
| Line Items per Label | 5 (with option to extend across multiple pages) |
| Seller Address | 120 characters (customizable) |
| Buyer Address | 130 characters (customizable) |
| Notes Section | 100 characters (customizable) |

### How to Preview

All changes are reflected in a **real-time preview panel** within the dashboard. You can also use the print option to check the updated label before saving.

### Shiprocket Branding

By default, "Powered by Shiprocket" is **mandatory**. To remove it, raise a request through your KAM or Tech Support team.

### Courier-Mandatory Fields (Cannot Be Changed)

All required courier-level fields (barcodes, routing codes) remain intact — your changes are limited to visibility, branding, and formatting only.

### Label Size

- Default: **6×4 inch** label for most couriers.
- **Bluedart:** A4 size label.
- **International orders:** Fixed A4 size.

### Field Availability by Courier

| Field | Exception |
|---|---|
| Order value on label | Bluedart, Delhivery, Wow, DotZot, Professional, Gati, Shadowfax — cannot be hidden |
| Mobile number on label | Ekart and Bluedart — cannot be hidden |
| Address on label | Ekart and Bluedart — cannot be hidden |
| Product name on label | Only available for XpressBees and Ekart |

### Applying Customizations

Once saved, label settings automatically apply to all **future shipments** from your account, across all channels.

**Channel-specific settings:** Currently not available. Settings apply account-wide. Contact Account Manager for channel-specific requirements.

### Additional Notes

- Changes apply to shipments created **after** saving the settings — not retroactively.
- You can edit and save changes anytime. Real-time preview helps verify before finalizing.
- Currently covers **shipping labels only**. Invoice customization is under development.
- No technical knowledge required — point-and-click with live preview.
- Settings are saved even after logout.

## How to Do on Seller Panel

### Access Label Settings

Settings > Company Setup > Labels (or Settings > Change Label).

### Customize and Save Label

1. Go to Settings > Company Setup > Labels.
2. Preview the current label.
3. Toggle fields ON/OFF (branding, seller/buyer address, barcodes, SKUs, etc.).
4. Adjust font size and style for available sections.
5. Set character limits for Address and Notes if needed.
6. Use the **real-time preview panel** to verify changes.
7. Click **Save** to apply to all future shipments.

### Download Labels

**Single order:** Orders > More (...) > Download Label.

**Bulk orders:** Select orders > Bulk Actions > Download Label.

**Bulk via scanning:** Orders > Ready to Ship > Scan icon > Paste AWBs > Choose format > Download Manifest/Labels.

### Change Label Format

Settings > Company Setup > Labels > Select format (A4, 4×6 inch, etc.) > Save.

### Remove Shiprocket Branding from Label

Contact your KAM or Tech Support team to raise a request. This cannot be done independently.

## Product Info

### What is Shiprocket Sense?

Shiprocket Sense is an advanced analytics and business intelligence API platform offering data-driven insights to e-commerce businesses. It helps sellers make smarter decisions using predictive analytics built on Shiprocket's logistics data.

**Website:** https://www.shiprocket.in/sense/

**Sign up:** https://console.shiprocket.in/login/

### Available APIs

| API | What it does |
|---|---|
| **RTO Risk** | Analyzes COD orders and predicts RTO risk along with RTO reason insights. Powered by Risk AI on 4.8 billion data points. |
| **Pincode Lookup** | Instantly converts pincodes to detailed location data — free to use. |
| **Pincode Serviceability** | Verifies delivery serviceability and provides EDD based on package specifics — free to use. |
| **Address Quality** | Analyzes address completeness, provides a quality score and risk assessment to reduce failed deliveries. |
| **Address Confirm** | Validates addresses by matching them against past delivery records. |
| **Order Risk** | Uses machine learning to analyze transaction history, customer behavior, and fraud patterns. |
| **Product Category** | Automatically categorizes products based on names, features, or descriptions. |

### Error Codes

| Code | Meaning |
|---|---|
| 200 | Successful. Check `"success"` field — if `true`, data is in the `"data"` field. |
| 401 | Invalid credentials or token. Check Authorization header. |
| 404 | URL not found. Check the requested URL. |
| 405 | Invalid data or method. Check credentials and syntax. |
| 422 | Missing or invalid data. Check `"invalid_attributes"` in response. |
| 500 | Internal server error. Verify correct syntax. |

### Required Headers

```
Content-Type: application/json
Authorization: Basic {{token}}
```

## Product Info

### Buyer Experience Overview

The Buyer Experience section allows sellers to manage how buyers interact with the post-purchase journey, including:
- Branded tracking page (Brand Boost)
- Return and exchange policy
- Quality Check (QC) settings
- Refund settings and modes
- Delivery Boost (AI-based NDR communication)

### Post-Purchase Tracking Page

Sellers can customize their tracking page with:
- Brand logo
- Social media links (Instagram, YouTube, etc.)
- Promotional banners
- Product recommendations

### Return & Exchange Policy Settings

| Setting | Description |
|---|---|
| Allow returns from tracking page | Toggle ON to let buyers initiate returns directly |
| Allow exchanges from tracking page | Toggle ON to let buyers initiate exchanges |
| Exchange window | Set between 1–30 days |
| Auto-approve exchange request | Auto-approves incoming exchange requests |
| Auto-assign return order | Auto-assigns courier to return based on reason |
| Auto-assign forward order | Auto-assigns forward (exchange) courier at a specified status |

### QC Parameters for Returns

- Product Name
- Product Image
- Size
- Brand Name
- Colour
- Unique Product Identifier (Serial No. / IMEI) — for electronics

### Refund Modes

| Mode | Description |
|---|---|
| Manual Refund | Seller processes manually; enters UTR number |
| Back to Source | Refunded to original payment method (prepaid only) |
| Store Credit | Discount coupon shared with buyer |
| Instant Refund | UPI, Bank Transfer, or Payout Links; auto-refund supported |

### Shipcoin (Buyer Rewards)

- Buyers earn 1% of order value OR 250 Shipcoins per fulfilled order (whichever is minimum).
- Validity: 12 months from date of credit.
- Shipcoins are credited on each order delivery.
- Used to avail gifts and rewards from leading brands via partner payment gateway.
- Not transferable to any other member.
- Sellers do not have visibility — Shipcoins are for buyers only.

## How to Do on Seller Panel

### Activate Brand Boost (Tracking Page)

**Buyer Experience > Tracking Page > Activate Now**

### Customize Tracking Page

**Buyer Experience > Tracking Page** — upload brand logo, add social media links, set themes, add promotional banners.

### Allow Returns from Tracking Page

**Buyer Experience > Brand Boost > scroll to bottom > Toggle ON "Allow Buyers to initiate return of products directly from the tracking page" > Save**

### Configure Return & Exchange Policy

**Buyer Experience > Return & Exchange Policy**

- **Return Settings:** Enable/disable returns, set return window.
- **Exchange Settings:** Enable/disable exchanges, set auto-approval, define exchange window.
- **QC Settings:** Enable/disable QC, choose QC parameters.

### Enable QC for Returns

1. Go to **Buyer Experience > Return & Exchange Policy > QC Settings**.
2. Toggle **Enable QC** to ON.
3. Select applicable QC parameters.
4. Click **Save**.

### Set Up Refund Settings

**Settings > Return Settings > Refund Settings**

1. Toggle on refund for COD orders (allows buyers to enter bank details).
2. Toggle on refund for Prepaid orders.
3. Choose refund modes: Store Credits, Back to Source, Manual bank details.
4. Set auto-refund trigger: at Picked Up, Delivered, or Acknowledged status.
5. Click **Submit**.

### Add QC Products

**Single Product:**
1. Go to **Returns & Refunds > QC Settings**.
2. Click dropdown (top right) > **Add Product**.
3. Fill: Quality Check Status, parameters (Product Name, SKU, Image, Brand, Colour, Size), Unique Product Identifier.
4. Click **Add Product**.

**Bulk QC Products:**
1. Go to **Returns & Refunds > QC Settings**.
2. Click dropdown > **Bulk**.
3. Upload the filled template file.

### Deactivate Brand Boost

**Buyer Experience > Tracking Page > Click here to deactivate > Select reason > Deactivate**

## Mobile App — How to Do

### Check Shipcoins (Mobile)

1. Open the **myShiprocket app**.
2. Tap the **top right icon** (profile/coins icon).
3. Tap on **coins** to see details and transaction history.

### How Buyers Use Shipcoins

1. Buyer chooses **Pay with Reward** during payment.
2. Enters OTP sent to device and submits.
3. Points are applied and discount is reflected on the order.

## Product Info (Enriched)

### WhatsApp Communication

Shiprocket sends WhatsApp order updates to buyers at the following statuses: Packed, Picked, Shipped, Out for Delivery, Arriving Early, Delivery Delayed, and Delivered.

**Benefits:**
- Keeps buyers updated in a non-intrusive manner.
- Reduces post-purchase anxiety.
- Improves read rate to 94%.
- Reduces customer queries by 30%.

**Charges:** Deducted from wallet — no buy-now-pay-later model.

**Demo:** Value Added Services > Notify > Try Demo > Enter phone number with active WhatsApp > Get Demo Message.

**Deactivate:** Value Added Services > Notify > Find deactivate option > Select reason > Deactivate.

### SMS Header

SMS header for buyer notifications **cannot be changed**. Current notifications are sent for: Picked, Shipped, Out for Delivery, etc.

### DND Numbers

If a buyer has DND enabled, they cannot be reached via the in-built customer connect module. Contact them through another medium.

### Buyer Communication Settings Path

Settings > Value Added Services > Notify > Email & SMS Communication tab.

### Email Template Customization

- 3 pre-defined templates available.
- Can create custom templates (Pro and Enterprise plans only).
- Click Edit on the Custom template to design as required.
- Font, layout, branding elements can be customized.

### Seller Notifications

Multiple emails and mobile numbers can be added for seller notifications:

Settings > Notifications > Seller Notifications > Add emails and phones for Business-related and Operations-related notifications.

To disable all seller notifications — contact support team.

### Tracking Page Analytics (Dashboard)

Dashboard > Tracking Page tab:
- **Unique Visitors** — distinct individuals who visited the tracking page.
- **Views** — total number of times tracking page was loaded.
- Filter by date range (date picker top-left below header).

### Return Initiation from Tracking Page

To allow buyers to initiate returns directly from the tracking page:

Buyer Experience > Brand Boost > Scroll to bottom > Toggle ON "Allow Buyers to initiate return directly from tracking page" > Save.

### Exchange from Tracking Page

Buyer Experience > Return and Exchange Policy > Exchange Settings > Toggle ON "Allow buyers to initiate an exchange directly from the tracking page" > Save.

## Product Info

### API Integration

- **API Docs:** https://apidocs.shiprocket.in
- **Token validity:** 10 days — must be regenerated every 10th day.
- Registered email **cannot** be used as API user; use a separate email.
- Bulk order uploads via API are **not supported**.

**Generate API:** Settings > API > Configure > Create API User > Fill details > Generate

### Channel Integration

Supports: Shopify, WooCommerce, Magento, Amazon (self-ship orders only), Unicommerce, and more.

- Auto-status sync: every **30 minutes**.
- Manual sync: click **Sync Orders** button on Orders page.

**Add channel:** Seller Panel > Channels > Add New Channel > Integrate > Follow steps

**Tech callback TAT:** 24 hours.

### Common Channel Sync Issues

| Channel | Common Issue |
|---|---|
| Amazon | Only self-ship orders synced |
| Shopify | Only unfulfilled/unshipped orders; check Pull Order Tag settings (if tags are set, they restrict which orders sync) |
| WooCommerce | Legacy API must be enabled; REST API must have read/write access |
| Others | Check connection status, account suspension, order sync settings |

Order changes on seller's website after sync do **not** auto-update on SR panel. Manual updates possible if status is "New."

### Unicommerce — Access Forbidden Error

Seller is using registered email instead of API user credentials. Guide seller to create an API user in the panel and use those in Unicommerce.

### ZOP (for Shopify)

WhatsApp-powered features: automatic order sync, status sync, catalog/inventory sync, auto-refund, abandoned cart messages.

| Contact Type | Action |
|---|---|
| ZOP seller | Handle at CC; deny product listing requests |
| Buyer | Route to support@zop.in or 9953253070 |
| Shopify sync issue | Raise ticket to Tech Support with ZOP tag |

### SR Support Portal

- Ticket creator name shows as "Abhinav Singh2"; source = "portal."
- Tickets go to respective Group, not Agent.
- Only ACR and NFCR cases appear on seller panel — FCR cases are not visible.
- If seller is unsatisfied with FCR resolution, they get one option to call CC.

### Shiprocket ONDC

ONDC = Open Network for Digital Commerce (Government of India initiative).

**Support helpline:** 8860542050 | **Email:** supportondc@shiprocket.com

**Eligible channels for onboarding:** Shopify, WooCommerce, Unicommerce only. Personal websites — soft denial.

**Remittance:** D+7 for all shipments. If TAT crossed — create NFCR ticket to ONDC Customer Support.

TAT: 2–3 days.

## SOP — Channel Integration Issue

1. Check channel connection status in ICRM.
2. Ask seller to reauthenticate from panel (Shopify/Amazon).
3. For WooCommerce — verify Legacy API enabled, REST API has read/write access.
4. Assign to **Customer Support Tech Desk**. TAT: 24 hours.

## How to Do on Seller Panel

### Connect a New Channel

Channels (left menu) > Add New Channel > Select channel > Integrate > Follow steps.

### Re-authenticate Shopify

Setup and Manage > Channels > Select Shopify > Re-authenticate > Log in > Authorize.

### Sync Orders Manually

Orders page > click **Sync Order** button (top right). Triggers immediate sync from all integrated channels.

### Generate API Token

Settings > API > Configure > Create API User > Fill details (separate email required) > Generate.

Regenerate every 10 days.

### Configure Shopify Pull Settings

On the Shopify channel settings, ensure "Pull Prepaid orders with Tags" and "Pull COD orders with Tags" are NOT set unless intentional — these restrict which orders sync.

### Create Support Ticket from Panel

Help & Support > Create Ticket > Select Category (Technical Support) > Select Sub-Category > Enter AWB/Order ID if applicable > Create Ticket.

## Mobile App — How to Do

### Connect Shopify (Mobile)

1. Three-dots menu at bottom > **Channel Integration**.
2. Tap **+** to add a new channel.
3. Enter store URL > **Connect to Shopify**.
4. Log in to Shopify > tap **Install App**.
5. Edit channel settings as needed.

> Other channels (non-Shopify) require the web panel.

### Sync Orders (Mobile)

From Orders screen, tap **Sync Order** to manually trigger sync.

### API Token Management

Requires web panel — not available on mobile app.

## Amazon Shipping (1 kg, 2 kg, 5 kg)

### Eligibility

- GSTIN invoicing enabled in Shiprocket settings.
- Prepaid orders only (COD coming soon).
- Order value below Rs. 50,000.
- Dimensions: max 70x70x45 cm.
- Weight: min 0.5 kg, max 18 kg.
- Non-hazmat items only.
- Surface mode; thermal printing labels required.

### Pickup Slots

| Shipment Created | Pickup Window     |
| ---------------- | ----------------- |
| Before 9:45 AM   | 10:00 AM – 1:00 PM|
| Before 12:45 PM  | 1:00 PM – 4:00 PM |

_Pickups are auto-scheduled — seller does not need to generate pickup manually._

## Multi-Packet Shipment (MPS)

Used for shipping multiple packages to the same recipient.

- Not plan-restricted.
- Available for **Delhivery** only.
- Manual orders only — does not work for channel integration orders.

### Specifications

| Spec          | Limit           |
| ------------- | --------------- |
| Max packages  | 20 (1 Master + 19 Child AWBs) |
| Max weight    | 1,000 kg        |
| Pickup        | Next day only   |

After training completion, MPS activation request is raised by the trainer to LOPS.

## Split Shipment

Allows partial fulfillment based on available stock.

**Enable:** Settings > Shipment Features > Split Shipment > Toggle ON

Works for **manual orders only**.

### Steps to Create a Split Shipment Order

1. Create order manually via Add Order.
2. Go to Processing tab > Select order > Ship Now.
3. Select products to ship in the popup.
4. Update package weight.
5. Click **Split and Ship Now**.
6. Choose courier partner.
7. You are redirected to Processing tab for remaining partial shipment.

## India Post

### Two Modes

- **Speed Post (Air):** Up to 2 kg.
- **Business Parcel (Surface):** 2 kg and above (minimum weight 2 kg).

### Key Information

| Feature          | Detail                              |
| ---------------- | ----------------------------------- |
| PIN code coverage| 19,000+                             |
| RTO charges      | Not applicable                      |
| COD              | Available (cash mode only)          |
| Pickup cut-off   | 10:00 AM                            |
| Pickup window    | 11:00 AM – 7:00 PM                  |

### Claim Liability

| Mode            | Liability                                                       |
| --------------- | --------------------------------------------------------------- |
| Speed Post      | Double the Speed Post freight charges or Rs. 1,000 (whichever less)|
| Business Parcel | Rs. 500 or actual value of lost / damaged content (whichever less)|

### SLA (excluding Sundays and National Holidays)

| Mode            | Zone A | Zone B | Zone C | Zone D | Zone E |
| --------------- | ------ | ------ | ------ | ------ | ------ |
| Speed Post      | 2 days | 4 days | 3 days | 5 days | 5 days |
| Business Parcel | 3 days | 6 days | 5 days | 7 days | 7 days |

### Zone Reference

| Zone | Description                          |
| ---- | ------------------------------------ |
| A    | Within City                          |
| B    | Within State                         |
| C    | Metro to Metro                       |
| D    | Rest of India (excluding NE and J&K) |
| E    | North East and J&K                   |

### Activation

- Company ID

## Rocket Express Services

Shiprocket's own pickup fleet (**Delhi only**) for SMB sellers.

- FE scans shipment on spot — status changes to "Picked Up" immediately.
- Shipments handed over to the originally assigned courier at the pickup hub.
- Seller can verify FE identity via the App (AWB number visible on FE app).

For sellers outside Delhi — inform that the service is currently unavailable.

## E-Way Bill

Required for shipments with value over Rs. 50,000.

### Upload at Pickup Generation

Seller Panel > Generate Pickup popup > Upload E-Way Bill number, PDF, and invoice.

### Upload from Ready to Ship Screen

Orders > Ready to Ship > Upload Eway Bill button.

### For RTO Shipments

Orders > Shipments > RTO > Upload Eway Bill.

**E-Way Bill validity:** 7 days.

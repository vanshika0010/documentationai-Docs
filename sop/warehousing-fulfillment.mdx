## Overview

Shiprocket Fulfillment is an e-commerce fulfillment solution offering order fulfillment, warehousing, catalog, and inventory management through tech-enabled fulfillment centers across India.

## Available Cities

Fulfillment service is available in **14 cities** across **45 warehouses**:

- Delhi
- Gurugram
- Jaipur
- Lucknow
- Ludhiana
- Faridabad
- Bangalore
- Chennai
- Hyderabad
- Kolkata
- Guwahati
- Mumbai
- Ahmedabad
- Surat

## Key Requirement

Seller must have the **GST number for the specific state** in which they want to store products in a Shiprocket warehouse.

## Arranging a Callback

If seller is interested — fill the Pipedrive form on SPARK and confirm the following mandatory details:

- Name
- Country code
- Number
- Address
- Product category
- Plan start date for shipments

Inform seller: Fulfillment expert team will call back within **24 hours**.

_Note: For warehouse inventory-related updates and information, email must be marked to shilpi.garg1@shiprocket.com (looping clients is mandatory on email)._

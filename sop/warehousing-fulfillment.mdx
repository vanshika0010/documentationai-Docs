## Product Info

### Overview

Shiprocket Fulfillment is an e-commerce fulfillment solution offering order fulfillment, warehousing, catalog, and inventory management through tech-enabled fulfillment centers across India.

### Available Cities (14 cities, 45 warehouses)

Delhi, Gurugram, Jaipur, Lucknow, Ludhiana, Faridabad, Bangalore, Chennai, Hyderabad, Kolkata, Guwahati, Mumbai, Ahmedabad, Surat.

### Key Requirement

Seller must have the **GST number for the specific state** in which they want to store products in a Shiprocket warehouse.

### Support Contact

For warehouse inventory-related updates and information, email must be marked to **shilpi.garg1@shiprocket.com** (looping clients is mandatory on email).

**SRF support email:** srfsupport@shiprocket.com

## SOP — Arranging Callback for Interested Sellers

If seller is interested in Shiprocket Fulfillment — fill the Pipedrive form on SPARK and confirm the following mandatory details:

- Name
- Country code
- Number
- Address
- Product category
- Plan start date for shipments

Inform seller: Fulfillment expert team will call back within **24 hours**.

## How to Do on Seller Panel

### Access Warehousing Features

Fulfillment-specific features (inventory management, order routing to fulfillment centers) are accessible from the left navigation menu under **Fulfillment** or **Warehousing**, depending on account activation.

### Enable Order Verification for Fulfillment

If using Shiprocket Fulfillment, enable Order Verification to verify orders before moving them to the warehouse:

**Settings > Additional Settings > Shipment Features > Order Verification > Toggle ON**

### Check GST Requirement

Before enrolling for fulfillment in a specific state, ensure you have registered GSTIN for that state. Upload it under:

**Settings > Company Setup > Billing-Invoice-GST**
